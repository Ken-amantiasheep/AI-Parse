# 软件使用说明（只保留生成 JSON）

这份说明只给普通使用人：  
你只需要上传 PDF，然后等待 JSON 输出即可。

---

## 1. 打开软件

1. 打开文件夹：`S:\Uploading Team\current\`
2. 双击：`start_gui.bat`
3. 等待窗口打开

---

## 2. 上传文件并生成 JSON

1. 在 `Company` 选择对应公司（按你平时业务选择）
2. 在 `Output Folder` 选择 JSON 保存位置（不会选就保持默认）
3. 把 PDF 拖进对应框，或点击框后选择 PDF 文件
4. 点击绿色按钮：`Generate JSON`
5. 等待完成，状态显示 `Success!` 就成功了

---

## 3. 到哪里找结果

- 生成完成后，JSON 会保存在你设置的 `Output Folder` 里
- 文件是 `.json` 结尾
- 成功后程序一般会提示是否打开该文件

---

## 4. 你只需要记住这句话

打开 `start_gui.bat` -> 上传 PDF -> 点 `Generate JSON` -> 等 `Success!` -> 去输出文件夹拿 JSON。
