# 软件使用说明（本地安装版）

这份说明给普通使用人。  
你只需要先安装一次，然后每次打开会自动检查更新。

---

## 1. 首次安装（只做一次）

1. 打开：`S:\Uploading Team\AI-parse\`
2. 双击：`install_client.bat`
3. 完成后桌面会出现 `AI-parse` 快捷方式（默认）

> 说明：软件将安装到本机 `%LocalAppData%\AI-parse`，不再直接在 S 盘运行。

---

## 2. 日常启动

1. 双击桌面 `AI-parse` 快捷方式（或运行本地 `start_ai_parse.bat`）
2. 启动器会自动检查 S 盘是否有新版本
3. 若有新版本，会弹窗问你是否更新
4. 点“是”会自动更新并继续启动；点“否”继续用当前本地版本

---

## 3. 上传文件并生成 JSON

1. 在 `Company` 选择对应公司
2. 在 `Output Folder` 选择 JSON 保存位置（不会选可保持默认）
3. 把 PDF 拖进对应框，或点击框后选择 PDF 文件
4. 点击绿色按钮：`Generate JSON`
5. 等待状态显示 `Success!`

---

## 4. 到哪里找结果

- JSON 保存在你设置的 `Output Folder`
- 文件后缀是 `.json`
- 成功后程序会提示是否打开文件

---

## 5. 记住这句话

打开桌面 `AI-parse` -> 需要时点更新 -> 上传 PDF -> 点 `Generate JSON` -> 等 `Success!` -> 去输出目录拿 JSON。
