# UAT 验收清单

## 功能验收

- [ ] Intact 样本至少 2 组可成功生成 JSON
- [ ] CAA 样本至少 2 组可成功生成 JSON
- [ ] GUI 可选择输出目录并正常写入文件
- [ ] 同名输出自动加 `(1)` 后缀

## 并发验收

- [ ] 3-5 人并发生成任务，均可完成
- [ ] 网关日志中有 request_id、company、docs、status

## 异常验收

- [ ] 网关停止时，客户端可给出明确错误提示
- [ ] INTERNAL_API_TOKEN 错误时返回 401
- [ ] ANTHROPIC_API_KEY 无效时返回可读错误
- [ ] 输出目录无权限时 preflight 失败

## 运维验收

- [ ] 网关服务开机自启动成功
- [ ] 升级到新版本后可正常运行
- [ ] 回滚到上一版本可在 5 分钟内完成
